COVID-19 Dataset analysis

About

This project was so important, I analze data from 2 different sources, 1. Covid 19 WHO, 2. Covid 19 NCDC. Data visualization done using Power BI, Tableau. Its's a semi-interactive dashboard which features a covid-19 data set I downloaded from WHO. first, I imported the data set to Visualization tool, After that, I imported the data to excel for data cleaning before importing it to Tableau, where I used the data to create this amazing dashboard!
This is more of a descriptive analysis which features:

1.Title: WHO Covid-19 Analysis
Data Visualization Tool: Tableau
Datasets Source: Quantum Analytics NG
- Insights generated
• Cumulative Cases
• Cumulative Deaths
• Deaths by Countries
• Cases by Countries

2.Datasets Source: Nigeria NCDC Website
Dataset:
Data about the pandemic event covid 19 in Nigeria
- Active Cases
- Total Confirmed Cases
- Total Deaths
- Total Discharged
- States by Confirmed Cases
- States by Total Deaths
